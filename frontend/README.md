# Frontend

This folder will contain the customer-facing dashboard.

## Suggested Stack

- React or Next.js
- Tailwind CSS
- Recharts or Chart.js for charts
- Authentication through Supabase or another auth provider

## Main Screens

### 1. Owner Dashboard
Shows:
- Daily revenue
- Top-selling products
- Slow-selling products
- Inventory warnings
- Restock recommendations
- Pending phone orders
- Daily AI summary

### 2. CSV Upload Page
Allows owners to upload sales data.

### 3. Approval Queue
Allows owners to approve, reject, or edit AI-created drafts.

### 4. Agent Activity Log
Shows what the AI agent recommended, drafted, or flagged.

## Frontend Tasks

- Build dashboard layout
- Build revenue chart
- Build product sales table
- Build inventory recommendation cards
- Build phone order draft cards
- Build approval queue buttons
