# Roadmap

## Phase 0: Planning

Goals:
- Define product scope
- Create repo structure
- Create documentation
- Assign intern tasks
- Prepare sample data

Deliverables:
- README
- Product requirements
- Architecture doc
- Agent safety rules
- Sample dataset

## Phase 1: Dashboard MVP

Goals:
- Build basic dashboard
- Upload sales CSV
- Display revenue and product trends
- Show top and slow products

Deliverables:
- CSV upload page
- Dashboard page
- Revenue chart
- Product table
- Sample data working end-to-end

## Phase 2: Inventory Recommendation Agent

Goals:
- Generate restock/prep recommendations
- Explain recommendations in simple language
- Store recommendation history

Deliverables:
- Recommendation logic
- Recommendation cards
- Daily summary agent
- Recommendation status field

## Phase 3: Approval Queue

Goals:
- Add owner control and safety workflow
- Allow approve/reject/edit actions
- Track audit log

Deliverables:
- Approval queue UI
- Approval API endpoints
- Agent activity log
- Status tracking

## Phase 4: Mock Phone Order Agent

Goals:
- Simulate phone order intake
- Create order drafts
- Display orders in dashboard

Deliverables:
- Mock order creation form
- Order draft cards
- Order approval workflow
- Transcript placeholder

## Phase 5: Real Voice Integration

Goals:
- Connect Twilio
- Allow AI to answer calls
- Create real order drafts from calls

Deliverables:
- Twilio number setup
- Voice webhook
- Call transcript storage
- Order draft creation from call

## Phase 6: POS and Payment Integrations

Goals:
- Connect real business tools
- Reduce manual CSV upload
- Support payment links

Possible integrations:
- Square
- Shopify
- Google Sheets
- Stripe

## Phase 7: Multi-Business SaaS

Goals:
- Support many businesses
- Add plans and billing
- Add onboarding workflow

Deliverables:
- Subscription plans
- Admin dashboard
- Business settings
- Role-based access control
